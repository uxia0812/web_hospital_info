## 📝 소개
> Biconnect에서 실사용을 위해 제작한 웹페이지입니다.

프리팅 입점 병원 정보가 검색엔진에 노출되기 위해 만든 웹페이지입니다.
상세 정보는 App을 다운받아 볼 수 있도록 유도하기 위해 필수적인 정보들로만 간략하게 볼 수 있도록 구성했습니다.

다음과 같은 내용을 작성할 수 있습니다.
- 프로젝트 소개
- 프로젝트 화면 구성
- 사용한 기술 스택
- 기술적 이슈와 해결 과정

<br />
